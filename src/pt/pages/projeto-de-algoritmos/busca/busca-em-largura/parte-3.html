3. Representação de Grafos

Um algoritmo deve operar sobre uma representação de dados de G.

3.1 Lista de Adjacência (Recomendada)

Cada vértice armazena uma lista de seus vizinhos:

Adj[u] = { v ∈ V ∣ (u, v) ∈ E }

Esta é a representação padrão no CLRS porque ela produz:

O(V + E)

de complexidade de tempo para a BFS.

3.2 Matriz de Adjacência

Uma matriz A ∈ {0,1}|V|×|V|:

A[u, v] = { 1 se (u, v) ∈ E; 0 caso contrário }

Essa representação leva a:

O(V2)

de complexidade para o percurso.

Intuição de engenharia: Para grafos esparsos (comuns em sistemas de engenharia), listas de adjacência são superiores.
