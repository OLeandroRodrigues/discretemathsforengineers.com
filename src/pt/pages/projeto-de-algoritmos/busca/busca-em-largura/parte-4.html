4. O Problema que a BFS Resolve

Seja s ∈ V um vértice fonte distinguido.

Queremos calcular:

4.1 Alcance (Reachability)

Quais vértices são alcançáveis a partir de s?

Reach(s) = { v ∈ V ∣ ∃ caminho de s até v }

4.2 Distância de Caminho Mínimo (Grafos Não Ponderados)

Defina δ(s, v) como o número mínimo de arestas em qualquer caminho de s até v.

Essa quantidade é chamada de distância no grafo.

A BFS calcula:

d[v] = δ(s, v)

para todos os vértices alcançáveis.

Importante: A BFS não minimiza pesos. Ela minimiza o número de arestas. Por isso funciona apenas para grafos não ponderados.
