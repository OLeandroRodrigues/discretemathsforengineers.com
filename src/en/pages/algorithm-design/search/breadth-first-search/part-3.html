3. Graph Representation

An algorithm must operate on a data representation of G.

3.1 Adjacency List (Preferred)

Each vertex stores a list of its neighbors:

Adj[u] = { v ∈ V ∣ (u, v) ∈ E }

This is the standard representation in CLRS because it yields:

O(V + E)

time complexity for BFS.

3.2 Adjacency Matrix

A matrix A ∈ {0,1}|V|×|V|:

A[u, v] = { 1 if (u, v) ∈ E; 0 otherwise }

This representation leads to:

O(V2)

time complexity for traversal.

Engineering insight: For sparse graphs (common in engineering systems), adjacency lists are superior.
